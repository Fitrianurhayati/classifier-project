WELLCOME TO CLASSIFICATION SYSTEM




Program dapat membantu Anda mengklasifikasikan judul skripsi menjadi 2 kelas yaitu Relata dan Sistem Cerdas




Petunjuk Penggunaan :


1. Pilih menu "Upload Data"

2. Pada menu "Upload Data" masukkan tahun dan pilih file yang hendak di analisis

3. Pada menu "Upload Data" setelah memilih file, maka akan menampilkan preview dari data tersebut

4. Pada menu "Analysis" akan menampilan hasil prediksi berupa Judul, Preprocessing dan Class

5. Pada menu "All Result" Anda dapat melihat hasil prediksi data berdasarkan tahun

6. Pada menu "Chart" Anda dapat melihat diagram kelas berdasarkan tahun
